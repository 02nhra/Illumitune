Illumitune is a exhibit at a kid's museum that has 8 infrared channels. When a channel is broken, a light turns on and a tone is played.

More information can be found at: http://www.sparkfun.com/tutorials/282

This repository contains all the firmware and design files for the various parts of the electronics. It also contains the logs from various dates. These logs show the usage information and are used for debugging and use trends.
